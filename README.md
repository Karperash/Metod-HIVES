# HIVES — метод группового принятия решений

## Курс: Методы принятия решений

---

## Цель проекта

Цель работы — изучить и прототипировать метод **HIVES** (Hierarchical Voting-based Evaluation System), который на основе социальных теорий и математической статистики позволяет объединять мнения участников группового принятия решений.

---

## Инструкция по запуску

1. Установите зависимости (Python 3.10+, NumPy):

```bash
pip install numpy
```

2. Запуск с тестовым файлом из статьи:

```bash
python main.py input.json
```

Или через обёртку:

```bash
python 1.py
```

3. Результаты выводятся в консоль: сырые и нормированные веса критериев (γ, γ'), итоговые баллы альтернатив и их ранжирование.

---

## Комментарии в коде: главные этапы

| Модуль | Этап | Кратко |
|--------|------|--------|
| **`json_input.py`** | Загрузка | Чтение JSON в `DecisionProblem`: альтернативы, критерии, ЛПР (матрица оценок A), эксперты (матрица весов W). |
| **`models.py`** | Модели | `Criterion`, `DecisionMaker`, `Expert`; `aggregated_performance()` — агрегированная матрица A; `weights_matrix()` — матрица W. |
| **`hives_method.py`** | HIVES | `score_bell_column` — функция Score Bell по одному критерию; `expand_by_influence` — «раздувание» W по влиянию экспертов (vote-casting); `hives` — веса γ и γ'; `hives_rank` — итоговые баллы и ранжирование альтернатив. |
| **`main.py`** | Сборка и вывод | Загрузка JSON → A, W → `hives_rank` → печать γ, γ', баллов и ранжирования. |

## Входные данные

Описание формата входных данных и постановки задачи — в базовой статье:

**https://www.sciencedirect.com/science/article/pii/S0957417422024903**

Формат JSON: `alternatives`, `criteria`, `dms` (матрица оценок альтернатив по критериям), `experts` (матрица весов критериев).

## Тестовый набор данных из базовой статьи

Файл **`input.json`** — данные, соответствующие таблицам 2 и 3 статьи (альтернативы A1–A15, 6 критериев, 7 экспертов).

## Выходные данные

Описание формата выходных данных (веса γ, γ', баллы, ранжирование) — в той же статье:

**https://www.sciencedirect.com/science/article/pii/S0957417422024903**

Результаты по умолчанию выводятся в консоль. Для сохранения в файл, например в папку **`outputs/`**:

```bash
mkdir outputs
python main.py input.json > outputs/result.txt
```

Папка **`outputs/`** — рекомендуемое место хранения всех результатов расчётов.
